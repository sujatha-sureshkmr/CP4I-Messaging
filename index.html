  1. NUS Mtech Projects

This section details the use case and guides for NUS MTech Projects.


Component Document Processing Module
View Details
The Intelligent EV Parts Component Retrieval System (IECRS), aimed at revolutionizing electric vehicle (EV) maintenance. This project addresses the challenge of efficiently retrieving sub-components from EV batteries by building a knowledge graph from existing resources such as text manuals and graphical images. Real-world statistics emphasize the growing complexity of EV batteries, highlighting the need for an intelligent system. The project unfolds in three phases: Team Project Proposal, Detailed Project Proposal, and Technical Paper & Report, with a focus on creating an interface that facilitates user prompts for efficient sub-component retrieval.

CP4I - Funds Transfer
View Details
In modern banking, the evolution of fund transfers thrives on an intricate integration powered by a suite of IBM solutions within Red Hat's Openshift environment. IBM Cloud Pak for Integration orchestrates this ecosystem, intertwined with IBM API Connect facilitating effortless mobile app transactions. Safeguarding the process, IBM MQ ensures secure queue management while IBM Event Streams drives real-time event processing, ensuring precision in fund transfer operations. This amalgamation culminates in a user-centric experience marked by instant notifications, representing a transformative era in banking, where innovation meets reliability, reshaping the landscape of financial transactions.
